# 插入

### 语法

```SQL
# 方式一：
insert into 表名(字段名，...)
  values(值1，...);
  
  
# 方式二:
insert into 表名
set  列名 = 值，列名 = 值



```


### 特点

```SQL
1、字段类型和值类型一致或兼容，而且一一对应


2、可以为 null 的字段
    ①：字段和值都省略
    ②：字段写上，值使用 null

3、不可以为 null 的字段，必须插入值


4、字段个数 和 值的个数必须一致


5、字段可以省略，但默认所有字段，并且顺序和表中的存储顺序一致
```


### 两种方式对比

```SQL

 1：方式一 支持插入多行,方式二不支持
# eg:
insert into 表名(字段名,...)
  values(值1,...),
  (值2,...),
  ...
  (值n,...);
  
 2：方式一支持子查询，方式二不支持
 # 机制是：先执行select的查询，再将查询的这个结果插入到表中
insert into 表名(字段名,...)
select  值1,... ;  # 注意这里的select 后面不仅是值，还可以查询语句
 
 
# 注意下面的两处字段名的个数也必须相同，可以为 null 依然是可以写 null
insert into 表名(字段名,...)
select  字段名,...
from 表名
where 筛选条件
...
  
```
